## ゲームの遊び方と特徴的な機能

### 基本の流れ：70箇所から宝石ピック🪨🔍

1. 6色の宝石から、賭けたい色と賭け金額を決定（複数色同時もOK！）
2. 宝石の中で当たりの倍率は1倍〜65倍まで（色ごとに変動）
3. 壁の70箇所の中から、最大20箇所を選んで宝石を“当てる”予想を立てる
4. 当たればそれぞれの倍率で配当獲得！

### 配当倍率（石の色と倍率）

| 石の種類 | ペイアウト倍率（賭け額の〇倍） |
| --- | --- |
| レッドストーン | 65倍 |
| ブルーストーン | 20倍 |
| グリーンストーン | 8倍 |
| パープルストーン | 4倍 |
| イエローストーン | 2倍 |
| グレイストーン | 1倍 |

### 特殊機能①｜プライズドロップ

- ゴンゾーが鍵を回すことで、選ばれた“ボーナスプライズ石”に追加の倍率や特典が付与される仕組み。
- +3〜+100の倍率アップ、2x〜10xの乗数、さらに再抽選チャンスなど、
- 最大10回まで再ドロップが可能で、大逆転のチャンスが劇的にアップ！

### 特殊機能②｜VR（バーチャルリアリティ）モード✨

- 業界初のVR対応ライブゲーム。
- 専用のVRヘッドセットとコントローラーがあれば、360度の没入体験が楽しめる。
- 日本では徐々に人気が高まっていて、先端技術好きのプレイヤーにおすすめ♪

## よくある質問

### Q1：無料でプレイできるデモモードはありますか？

- 一部のカジノでフリープレイが用意されていますが、公式にはデモモードがないことも多いです。
- ボーナスを使った無料プレイがメインのため、登録後にカジノのキャンペーンをチェックしてみてください！

### Q2：フリースピンやボーナスゲームはある？

- 通常のスロットのフリースピンはありません。
- 代わりにマルチプライヤーやプライズドロップ機能がゲームの魅力ポイントです✨

### Q3：日本円入出金時の手数料は？

- ベラジョンなど多くの日本向けカジノは**JPY対応で手数料が無料または格安**です。
- 詳しくは各カジノの入出金規約を必ず確認してください。

### Q4：日本で利用しても法律面で大丈夫？

- オンラインカジノの日本国内の法的状況は複雑ですが、Evolution Gamingは**マルタやジブラルタルなど複数国で適法ライセンス取得済み**。
