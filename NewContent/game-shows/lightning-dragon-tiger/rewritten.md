## 遊び方＆ゲームの流れ

ライトニングドラゴンタイガーはルールがシンプルなので初心者でもすぐに慣れますッ♪  
**ドラゴンとタイガー、どちらのカードが強いか当てるだけ！**  
まずは基本の流れを押さえておきましょう💡

### 1. ベットを決める

- ベット時間は約12秒間。チップをドラゴンかタイガー、それから「タイ（引き分け）」または「スーテッドタイ（同じスートの引き分け）」に賭けることもできます。  
- ベット金額は、カジノによって変わりますが日本のオンラインカジノでは約100円〜10万円くらいまで設定可能。  
- ベット額に手数料などの追加費用が発生するかは、利用カジノごとに異なるため事前に要確認です。  

### 2. ライトニングラウンド発動

- 賭けが締め切られたあとに「ライトニングカード」がランダムで複数枚選ばれ、2倍〜最大500倍までのマルチプライヤーが適用されます⚡  
- このマルチプライヤーは選ばれたカードに関係する勝利に付与され、通常の配当を一気にブーストしてくれる大きなチャンス！  

### 3. カードが配られる

- ディーラーがドラゴン側とタイガー側にそれぞれ1枚ずつカードを配ります。  
- カードの強さはA（最弱）→2→3→…→K（最強）の順で判断。  

### 4. 勝敗判定＆配当

- ドラゴンとタイガーのどちらのカードが大きいかで勝者決定。  
- 「タイ」や「スーテッドタイ」は特別配当対象。（当たる期待度は低いですが、その分配当も高め）  
- 勝った時は、ライトニングラウンドでかかったマルチプライヤーが適用されれば配当が大幅アップ！  

**🙋ここポイント**  
「タイの場合は賭け金が半分返却される」ルールなので安心ですねッ♪

## 日本向けの支払い方法・通貨対応

日本からのプレイにおいていちばん気になるのは、やっぱり**簡単で安全な入出金**ですよね！  

### JPY（日本円）対応

多くの国際オンラインカジノは日本円対応を進めていますッ♪  
Evolutionの提供するゲームを採用している以下のような日本向けカジノでは、**JPY建てでそのまま賭けられて為替リスクナシ**が多いです。  

| 支払い方法             | 特徴                                                                    |
| --------------------- | ----------------------------------------------------------------------- |
| クレジットカード（Visa、Master） | 即時入金可能。手数料はカジノにより異なるので要確認。                           |
| 電子ウォレット（ecoPayz、SticPay等）| 入出金ともに速く手数料も低め。日本人ユーザーも多く利用されています。          |
| 銀行振込（国内銀行対応）       | 着金まで数時間～数日かかる場合あり。特定カジノで対応。                             |
| 仮想通貨（BTCなど）          | 一部カジノで導入。法規制回避目的や入出金の匿名性重視に活用。                        |

---

### 取引の信頼性・処理速度

- 良質なカジノは**入金反映が数秒〜数分程度**と速く、不具合も少なめ。  
- 出金は基本的に数時間から数営業日で処理されますが、カジノや決済方法、本人確認（KYC）状況によって異なります。  
- KYC（本人確認）は日本の法律や国際ライセンスに準拠して必須で、本人書類（パスポートなど）を提出すると出金保留がスムーズに解除されやすいですっ♪

## ライブカジノの日本での法規制と安全性

日本ではカジノを含むギャンブルに関してかなり厳しい規制があります😳  
ただし、オンラインカジノは海外で運営されており、国内のオンラインギャンブル禁止法の直接対象外となっているケースが多いんです。  

### 公式ライセンスと運営場所

- Evolution社のゲームは主にマルタ（MGA）やジブラルタルなどの信頼できる運営ライセンスのもとで配信されています。  
- 日本向けのカジノは通常、このような国際ライセンスを持つ運営会社のサービスを利用。  
- 日本の広告規制に従い、露骨なギャンブル推奨や未成年者勧誘は法律違反にあたるため、多くのカジノはコンプライアンス遵守に力を入れています。  

### 税金の取り扱い

- 日本の法律ではギャンブルの勝ち金は「一時所得」に該当し、課税対象になる場合があります。  
- ただし海外オンラインカジノの収益申告義務や実際の税務処理は複雑なので、巨大な勝ち金や継続的に稼ぐ方は税理士等に相談したほうが安心です。
