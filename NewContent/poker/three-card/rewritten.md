## スリーカードポーカーとは？

![Casiko Happy](https://img.casinotsu.com/svg/jp/mascots/happy/casiko.svg)

スリーカードポーカーは、3枚のカードを使ってディーラーと勝負するカジノゲームです。

- 片手間で気軽に遊べるため、初心者でもすぐに取り組めます。
- ポーカーの駆け引きを楽しみながらも、ブラックジャックのようにディーラーとの対戦形式で分かりやすいのが特徴。
- ランドカジノはもちろん、オンラインカジノでもライブディーラー版やスマホ対応のゲームが充実しています。

## 基本ルールと進め方

ゲームの流れは以下の通りです：

1. **ベットを選ぶ**  
   - **Ante（アンティ）ベット**：ディーラーに挑戦する基本の賭け。
   - **Pair Plus（ペアプラス）ベット**：自分の手札だけでペア以上になるかを予想するサイドベット。
   両方に同時に賭けることも可能。

2. **カード配布**  
   プレイヤーとディーラーにそれぞれ3枚ずつ裏向きで配られます。

3. **プレイヤーの選択**  
   自分の手を見て、「フォールド（降りる）」か「プレイ（賭けを続ける）」のどちらかを決めます。  
   - フォールドした場合、アンティベットは没収されます。ペアプラスベットも負け扱い（没収）です。  
   - プレイする場合は、アンティと同額のプレイベットを追加します。

4. **ディーラーの手公開・判定**  
   ディーラーはクイーンハイ以上でなければプレイヤーと勝負できません。  
   - ディーラーがクイーンハイ未満の場合、アンティは1倍払い戻し、プレイベットはプッシュ（返却）となります。  
   - ディーラーがクイーンハイ以上の場合、プレイヤーとディーラーの役を比較し、強い方の勝利となります。引き分けはベット返却。

5. **ボーナス支払い**  
   プレイヤーの手がストレート以上ならば、役に応じたボーナス配当が付きます。

## アンティ／プレイ（Ante/Play）ベットの仕組みとペイアウト

アンティ／プレイベットはゲームの基本となる掛け金です。手役の強さに応じたペイアウトは以下の表のとおりです：

| 手役 | 配当（アンティベットに対して） |
| --- | --- |
| ストレート | 1:1 + 5:1（ボーナス） |
| スリーカード | 1:1 + 4:1（ボーナス） |
| フラッシュ | 1:1 + 3:1（ボーナス） |
| その他（クイーンハイ以上） | 1:1のみ |
| ディーラーの役不成立 | アンティ 1:1, プレイ プッシュ |

- プレイヤーの勝利でアンティ・プレイ両方の掛け金に1倍配当。  
- 強い役を持つとボーナスが加算されます。

## ペアプラス（Pair Plus）ベットの遊び方と配当

ペアプラスはディーラーに関係なく配当が決まる自己完結型ベットです。自分のカードが特定の役を満たせば払い戻しを受けられます。

| ハンド（役） | 配当例（カジノにより異なる） |
| --- | --- |
| ストレートフラッシュ | 40:1 |
| スリーカード（三枚同じ数字） | 30:1 |
| ストレート | 6:1 |
| フラッシュ | 4:1 |
| ペア | 1:1 |

- 配当は通常上記の範囲で設定されていますが、カジノによってやや異なることがあるため事前に確認をおすすめします。  
- フォールド時はペアプラスベットも没収されるので注意してください。

## 日本人におすすめのオンラインカジノと注意点

### 日本語対応とJPYサポートが重要！

- 多くの信頼できる国際オンラインカジノは**日本語インターフェース**を完備し、日本人プレイヤーが使いやすい環境を整えています。  
- また、**日本円（JPY）での入出金対応**は為替手数料の節約につながるので要チェック。  
- 入金・出金方法はクレジットカード、電子ウォレット（エコペイズ、スティックペイなど）、仮想通貨も一部は対応していますが、利用できる方法はサイトごとに異なります。

### モバイル対応・ライブディーラーの充実

- 多くのカジノがスマホやタブレットでも快適に遊べるモバイル版を用意しており、スリーカードポーカーのライブディーラーゲームも提供されています。  
- ライブバージョンは実際のスタッフとリアルタイム対戦できるので、ランドカジノの臨場感を味わいたい方にぴったりです。

### 信頼性を左右するライセンスと安全性

- 日本ではオンラインカジノの運営は国内で認可されていませんが、多くの優良サイトは**マルタ（MGA）やUK（UKGC）など、国際的に信頼性の高いライセンスを取得**しています。  
- こうした認可はゲームの公平性、個人情報の保護、セキュリティ面での安心材料となります。  
- 不安な場合は、ライセンス情報を必ずチェックし、評判の良いカジノを選びましょう。

### 出金時間と手数料

- 入金は即時反映が多いですが、出金は手続きや選択した方法によって数時間から数日かかることがあります。  
- JPY対応カジノなら通貨変換手数料がかからず効率的ですし、手数料もカジノによっては無料のところもあります。  
- 出金の際は本人確認（KYC）が必須となる場合が一般的なので、事前に書類提出を済ませておくとスムーズです。
