# 1. ペイズの日本円対応と銀行連携について

### 1-1. 本当に「日本円」で使えるの？🤔

歴史的にはPayzのカード発行は日本円での対応が限定的でしたが、**2025年時点でPayzアカウント自体は日本円対応を公式で明言**しています。  
これにより、オンラインカジノへ入出金する際、わざわざドルやユーロに換金しなくても良くなり、余計な手数料が減ってお得です🥳

### 1-2. 日本の銀行・カードからのチャージ方法

- **銀行振込・ネットバンク連携**  
  主要都市銀行、ネット銀行からPayzへのチャージが可能。金融機関によってはリアルタイム入金OK。  
  手数料や着金反映時間は銀行により異なるため事前確認がおすすめ。

- **クレジットカード・デビットカード**  
  VISA/Master等の日本発行カードも利用可能→Payzアカウントに即時反映されて便利！  
  ただし、一部カードは海外利用扱いとなりカード会社手数料の発生に注意。

> 💡 ワンポイント  
> 事前にPayzアカウントへチャージが必要なので、オンラインカジノで直接カード決済は不可（Payzへチャージ→カジノへ送金という形）。

# 3. Payzの本人確認（KYC）とアカウントステータスについて👤

ペイズでは、本人確認は必須ではありませんが、**認証完了しないと取引限度額が大幅に制限されます。**

| ステータス       | 説明                                                   |
|-------------|------------------------------------------------------|
| 未認証（ブロンズ） | 限度額が低めで入出金制限大。手数料もやや高め。                     |
| 認証済み（シルバー以上） | 身分証明書（パスポート等）＋住所証明（公共料金請求書など）を提出。限度額大幅UP。 |

### 3-1. 本人確認に必要な書類

- **パスポート**（顔写真ページのコピーまたは写真）  
- **住所確認書類**（政府発行の書類、公共料金の請求書等）  
- **その他、状況に応じて追加書類要求も（例：クレジットカード表裏）**

本人確認はカスタマーサポートから案内されることが多いです😊  
これが済めば「高額の入出金OK」「手数料が低減」「よりスムーズな資金移動」が可能です！

# 4. 入出金の具体的な手続き＆処理時間⌛

### 4-1. ペイズからオンラインカジノへの入金の流れ

1. Payzに先に日本円でチャージ済みか確認。  
2. オンラインカジノの入金ページで決済手段に「Payz」を選択。  
3. 入金額入力後、PayzのIDとパスワードを使って認証。2段階認証（2FA）も要求される場合あり。  
4. 入金完了し、すぐにカジノアカウントへ反映されるケースが多い（数秒〜数分）。  

### 4-2. 出金の流れ

1. 出金ページでPayzを選択、アカウントIDを入力。  
2. 出金額指定後、カジノが処理。処理が完了するとPayzアカウントに着金。  
3. Payzから日本の銀行口座やカードへ送金手続き。  
4. 銀行口座受取までは1〜3営業日（銀行の営業日に左右される）。  

> 🔔 **ポイント**  
> 出金申請前に同じPayzアカウントからの入金履歴が必要なことが多いです。詐欺防止のためのルールですので必ず覚えておきましょう！
